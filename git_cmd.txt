1: tạo project trên github xong lấy về local(work alone)

    git status (kiểm tra trạng thái file)
    git add . (thêm file)
    git commit -m "your message" (tạo nội dung commit)
    git push origin main

2: đẩy project từ local lên github 	
    tao 1 github repo
    git init
    git status
    git add .
    git commit -m "your message"
    git remote add origin https://......
    git push origin master

3: các câu lệnh khi làm việc teamwork
    git pull (kéo các thay đổi mới nhất về)
    git branch (kiểm tra xem có những nhánh nào)
    git checkout -b branch_name (tạo 1 nhánh mới có tên là branch_name)
    git conflict
